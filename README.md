# Clifford

A simple command-line tool for computing the real clifford algebras in signature (s,t) as matrix algebras over the normed division algebras ℝ, ℂ, ℍ.

## Dependencies
The script requires the scala intepreter.

**Arch Linux**

    pacman -S scala

## Example

    ./clifford.sh 3 1
